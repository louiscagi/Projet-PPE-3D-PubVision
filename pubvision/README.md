### Run Locally

To run **Pixology** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/louiscagi/Projet-PPE-3D-Publicis
```

Windows:

```bash
git clone https://github.com/louiscagi/Projet-PPE-3D-Publicis
```
